# Welcome to Lumina Docs

Lumina is an open-source React UI component library built on top of Tailwind CSS. It helps you build modern, responsive web interfaces faster, without reinventing common UI patterns every time.

All components are designed to be accessible, customizable, and easy to drop into real-world projects. Lumina works great with React-based frameworks like Next.js, React, Astro, and more.

## Getting Started

New to Lumina? Start here:

- **[Installation](/installation)** – Set up Lumina in your project in minutes

- **[Components](/components)** – Browse all available UI components

## Why Lumina?

Building consistent UI at scale is hard. Lumina removes friction by giving you production-ready components that are easy to understand and easy to adapt.

With Lumina, you can:

- **Build faster** – Skip repetitive UI work and focus on product logic.
- **Stay consistent** – Keep a clean, unified design system across your app.
- **Ship responsive UI** – Components work out of the box on all screen sizes.
- **Customize freely** – Tailwind utilities give you full control over styling.

## What You Get

Lumina is designed for real projects, not just demos.

- **Modern UI patterns** – Clean, practical designs based on current UI best practices.
- **Fully responsive components** – Mobile-first and desktop-ready.
- **Accessibility-first** – Built with accessibility considerations baked in.
- **Tailwind-powered** – Customize everything using familiar Tailwind classes.
- **Clear documentation** – Simple examples and usage guides for each component.

Lumina is designed to stay out of your way. Clean components, predictable structure, and full control when you need it.
