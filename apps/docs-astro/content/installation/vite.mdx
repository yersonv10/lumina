# Vite Installation

Follow these steps to set up Lumina in your React project using Vite.

<Steps>

<Step>
### Create project

Create a new React project using Vite with TypeScript:

```npm
npm create vite@latest my-app -- --template react-ts
cd my-app
npm install
```

</Step>

<Step>
### Add Tailwind CSS

Install Tailwind CSS and the Vite plugin:

```npm
npm install tailwindcss @tailwindcss/vite
```

Add the plugin to `vite.config.ts` and import Tailwind in your main CSS file:

```ts title="vite.config.ts"
import tailwindcss from "@tailwindcss/vite";
import react from "@vitejs/plugin-react";
import { defineConfig } from "vite";

export default defineConfig({
  plugins: [react(), tailwindcss()]
});
```

```css title="src/index.css"
@import "tailwindcss";
```

</Step>

<Step>
### Install Lumina

```npm
npm install @lumina/ui @lumina/icons
```

Add the theme to `src/index.css`, after `@import "tailwindcss";`:

```css title="src/index.css"
@import "tailwindcss";
@import "@lumina/ui/theme.css";
```

See [Global CSS Variables](/get-started/global-variables) for the full list of tokens this brings in.

</Step>

<Step>
### Use the Component

Use the component in your app:

```tsx title="src/App.tsx"
import { Button } from "@lumina/ui";

function App() {
  return (
    <div className="flex min-h-svh items-center justify-center">
      <Button>Click me</Button>
    </div>
  );
}

export default App;
```

</Step>

</Steps>
