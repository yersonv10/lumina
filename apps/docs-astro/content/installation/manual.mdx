# Manual Installation

Follow these steps to set up Lumina in your existing project.

<Steps>

<Step>
### Install Tailwind CSS

Ensure your project has [Tailwind CSS](https://tailwindcss.com/docs/installation) installed and configured.

</Step>

<Step>
### Install Lumina

```npm
npm install @lumina-kit/ui @lumina-kit/icons
```

Add the theme to your global CSS, after your `@import "tailwindcss";`:

```css
@import "tailwindcss";
@import "@lumina-kit/ui/theme.css";
```

See [Global CSS Variables](/get-started/global-variables) for the full list of tokens this brings in.

</Step>

<Step>
### Use a Component

Import and use the component in your application:

```tsx
import { Button } from "@lumina-kit/ui";

export default function Example() {
  return <Button>Click me</Button>;
}
```

</Step>

</Steps>
