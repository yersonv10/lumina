# Next.js Installation

Follow these steps to create and set up Lumina in a fresh Next.js project.

<Steps>

<Step>
### Create project

Create a new Next.js project with TypeScript and Tailwind CSS:

```npm
npx create-next-app@latest --ts --tailwind --import-alias "@/"
```

Follow the prompts to complete the project setup.

</Step>

<Step>
### Install Lumina

```npm
npm install @lumina-kit/ui @lumina-kit/icons
```

Add the theme to `src/app/globals.css`, after your `@import "tailwindcss";`:

```css title="src/app/globals.css"
@import "tailwindcss";
@import "@lumina-kit/ui/theme.css";
```

See [Global CSS Variables](/get-started/global-variables) for the full list of tokens this brings in.

</Step>

<Step>
### Use the Component

```tsx title="src/app/page.tsx"
import { Button } from "@lumina-kit/ui";

export default function Home() {
  return (
    <div className="flex min-h-svh items-center justify-center">
      <Button>Click me</Button>
    </div>
  );
}
```

</Step>

</Steps>
