# React Router Installation

Follow these steps to set up Lumina in your React Router project.

<Steps>

<Step>
### Create project

Create a new React Router project:

```npm
npx create-react-router@latest my-app
cd my-app
```

Follow the prompts to complete the project setup.

</Step>

<Step>
### Install Lumina

```npm
npm install @lumina-kit/ui @lumina-kit/icons
```

Add the theme to your global CSS, after your `@import "tailwindcss";`:

```css
@import "tailwindcss";
@import "@lumina-kit/ui/theme.css";
```

See [Global CSS Variables](/get-started/global-variables) for the full list of tokens this brings in.

</Step>

<Step>
### Use the Component

Import and use the component in your application:

```tsx title="src/routes/home.tsx"
import { Button } from "@lumina-kit/ui";

export default function Home() {
  return (
    <div className="flex min-h-svh items-center justify-center">
      <Button>Click me</Button>
    </div>
  );
}
```

</Step>

</Steps>
