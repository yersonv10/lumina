# Installation

To get started with Lumina, you first need a React project with [Tailwind CSS 4](https://tailwindcss.com/) set up.

## Quick Start

Lumina is a regular npm package — no CLI required.

```npm
npm install @lumina-kit/ui @lumina-kit/icons
```

Then add the theme to your global CSS, after your `@import "tailwindcss";`:

```css
@import "tailwindcss";
@import "@lumina-kit/ui/theme.css";
```

See [Global CSS Variables](/get-started/global-variables) for the full list of tokens this brings in.

## Pick Your Framework

Start by selecting your framework of choice. Then follow the instructions to install the dependencies and structure your app.
