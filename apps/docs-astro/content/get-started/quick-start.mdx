# Quick Start

Lumina is a regular npm package — install it and import components like any other library. This guide covers setup for a project that already has React and Tailwind CSS 4.

**Starting from scratch, or using Astro, Next.js, Vite, TanStack Start, or React Router?** Head to [Installation](/installation) and pick your framework — it walks through scaffolding a project before you get here.

<Steps>

<Step>
### Install the packages

```npm
npm install @lumina/ui @lumina/icons
```

`@lumina/ui` ships every component; `@lumina/icons` is a separate package with 700+ icon components — install it only if you use icons.

</Step>

<Step>
### Add the theme

Add this line to your global CSS, after your existing `@import "tailwindcss";`:

```css title="src/app.css"
@import "tailwindcss";
@import "@lumina/ui/theme.css";
```

This brings in the design tokens every component relies on (colors, radius, shadows) and tells Tailwind to scan `@lumina/ui`'s compiled output for utility classes, since Tailwind never scans `node_modules` by default. See [Global CSS Variables](/get-started/global-variables) for the full list of tokens. Prefer dark mode? Import `@lumina/ui/theme-dark.css` instead.

</Step>

<Step>
### Import and use components

```tsx filename="src/app/page.tsx"
import { Button } from "@lumina/ui";

export default function Home() {
  return (
    <main className="flex min-h-screen flex-col items-center justify-between p-24">
      <Button variant="primary">Hello Lumina!</Button>
    </main>
  );
}
```

</Step>

</Steps>
